## Features

- **Send securely** shares the selected files over a private link that's copied to your clipboard and expires on its own
    > Files transfer straight from your Mac, so a link only works while you're sharing it
- **Transfers** panel on the Send button shows each active link's download count and lets you copy, reschedule, or stop it
- A notification tells you when someone finishes downloading a file you sent
- Send securely straight from the right-click menu, using your default expiration
- **Custom shortcuts**: rebind any toolbar action's keyboard shortcut in Settings, under Shortcuts
- Hold `⌘` to reveal every action's shortcut right on its button

## Improvements

- **Redesigned action toolbar**: actions are grouped into sections with the less-used ones moved into a `⋯` menu, so the row stays clean and glanceable
- Restyle the toolbar to taste: icon-and-text, text-only, or icon-only buttons, regular or compact spacing, and optional dividers and background
- Decide where each action lives, in the bar, the `⋯` menu, or hidden, from Settings

## Fixes

- Open-with app hotkeys no longer override action shortcuts like `⌘⌥C` for Copy to…
    > Apps in the Open With bar skip letters already taken by a built-in action
